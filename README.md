# ccr-time-tracker-next

## Inicializando projeto

Instale as dependendências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) com seu browser para ver o resultado.
